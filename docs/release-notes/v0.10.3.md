# Release Notes: antemortem 0.10.3

## Summary

- Package version: `0.10.3`
- Scope: CI-status integrity for README-linked companion docs; release surface version metadata.
- Runtime behavior: unchanged; the only runtime source edit is the package `__version__` string.

## Fixed

- Re-pinned `README.md` repository links from the frozen `v0.10.2` tag to `v0.10.3`. At `v0.10.2`, the linked trust model, toolkit positioning, and CLI example gallery docs were last modified by a commit whose CI run predated a trust-check fix, so each file page rendered a failing-CI badge. The `v0.10.3` tag includes a passing-CI commit for those docs.
- Added a README backlink footer to the trust model, toolkit positioning, and CLI example gallery docs.

The `v0.10.3` GitHub tag must exist before uploading to PyPI. Until that tag is pushed, tag-pinned links in the PyPI README can return 404.
